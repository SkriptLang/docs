Syntax is Skript's most core component, powering the scripts you write.
First, this guide covers the general properties of all syntax, along with the process for defining them.
Next, it covers the standard types of syntax implemented by Skript.

:::tip
For beginners, these pages are intended to be read in order.
We introduce different types of syntax in order of approximate complexity.
We recommend using the navigation buttons at the bottom of each page to walk yourself through this guide.
:::
